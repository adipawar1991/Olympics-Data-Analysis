# Olympics-Data-Analysis

A data analytics project analyzing historical Olympic data with interactive dashboards built using Python, Pandas, Plotly, and Streamlit.

## Features
- Medal Tally Analysis
- Country-wise Performance
- Athlete Statistics
- Year-wise Trends
- Interactive Visualizations

## Tech Stack
- Python
- Pandas
- Plotly
- Streamlit

## How to Run
streamlit run app.py
