##  Project Overview

The Olympics Data Analysis project analyzes historical Olympic datasets to uncover patterns, trends, and insights such as:

- Medal tally by country
- Performance trends over years
- Athlete participation analysis
- Country-wise dominance
- Sport-wise medal distribution

The project is built with an interactive Streamlit dashboard for real-time filtering and visualization.

##  How to Run the Project Locally

### 1. Clone the Repository

```bash
git clone https://github.com/adipawar1991/Olympics-Data-Analysis.git
cd Olympics-Data-Analysis
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the Application

```bash
streamlit run app.py
```

The application will open in your browser.
